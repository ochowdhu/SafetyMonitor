\begin{abstract}
The trend towards increasing commercial-off-the-shelf (COTS) components in complex safety-critical systems is increasing the difficulty of verifying system correctness. 
Runtime verification (RV) is a promising technique to check executions of these systems, 
but most existing RV methods utilize some form of system or code instrumentation and thus are not designed to monitor potentially black-box COTS components.
%
Instead of accessing state through instrumentation, we explore monitoring a system's externally observable state, which can be obtained without affecting the target system.
This paper presents a framework for monitoring safety-critical systems comprised of black-box components. We present a broadcast-bus based monitor which checks bounded metric temporal logic specifications.
We have implemented an embedded, real-time CAN monitor utilizing our monitoring algorithm. The monitor has been evaluated on a bench CAN bus with replayed logs obtained from system testing of an autonomous research vehicle on which violations of system safety requirements were identified.
